Heat Equation
Displays a rectangular plane viewed with Orthographic camera. A RGB texture is created from a matrix of temperature values by applying a color map. The texture is applied to the material of the plane. Repeatedly the matrix is updated by replacing each pixel with the average of its neighbour pixels.

Magnification
Graphics cards have built-in support for nearest and bilinear interpolation of textures (magnification). Nearest gives a pixelated image when zooming in. Bilinear gives a smoother image.
When zooming out aliasing and moire patterns may be a problem. Graphics cards have built-in support for mipmapping. That is a down sampled texture is used when zoomed out (minification). Theory

PointCloud
Reads a cloud of lidar xyz points from an ascii file into the vertices of a Geometry. Sets the colors of the geometry based upon the z value of each vertex using a color map. Combine the geometry with a PointsMaterial into a Points object.

MeshMaterials
Shows the differences between the materials a Mesh may have.
Total illumination, I, is a sum: I = Ia + Id + Is, Ia: ambient light, Id: diffuse (non shiny) reflection, Is: specular (shiny) reflection.
BasicMaterial only takes into account ambient light, Ia, and has no shading.
LambertMaterial takes into account ambient and directional lights, Ia and Id, and has shading but is not shiny.
PhongMaterial takes into account account ambient light and directional lights, has shading and is shiny, Ia, Id, and Is.


Skybox
Skybox is a way to create realistic surroundings. A huge CubeGeometry is placed around the camera. For each face of the cube a MeshBasicMaterial with a separate texture is created. The texture is an image of the surroundings as seen from the center of the cube projected onto the face. A MeshFaceMaterial is created from an array of all the materials. This material is combined with the geometry into a Mesh.

DigitalElevationModel
Reads elevation from a 16 bit unsigned integer png into a texture. Call an ESRI map service to get a RGB png for the same area and read this into another texture. Pass both textures to the shaders using ShaderMaterial. Use the elevation texture to set z in the vertex shader. Use the color texture to set the color in the fragment shader. The result is a map draped onto an elevation model.
